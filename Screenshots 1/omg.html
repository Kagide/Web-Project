Datasets

In this project, we use a labeled dataset of URLs to train machine learning models that can detect whether a given URL is harmful or safe.

Source

The dataset was obtained from open-source security platforms and research repositories such as Kaggle and PhishTank. It includes both malicious and benign URLs, allowing for supervised learning.

Structure

Each URL is represented by several features that help identify suspicious patterns. Here are some of the key features:

  • URL: The full web address
  • Has IP Address: Whether the URL uses a direct IP address instead of a domain name
  • Length: Total number of characters in the URL
  • Number of Special Characters: Counts of characters like `-`, `_`, `@`, `?`, `=`, `%`, etc.
  • Presence of Suspicious Words: Words like ‚Äúlogin‚Äù, ‚Äúverify‚Äù, ‚Äúsecure‚Äù, etc.
  • Label: Target class (Malicious or Benign)
